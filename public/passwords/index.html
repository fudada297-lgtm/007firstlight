007 First Light — All Safe Codes, Door Passwords & Puzzle Solutions

Important: All passwords and codes in 007 First Light are fixed — they are the same on every difficulty level. You don't need to find them in-game if you already know them.

Chapter 8: All the Time in the World (Slovakia Hotel)

Laundry Room Door Code
1805
Found on a note in the Manager's Office desk, on the same floor as the front desk (restricted area).
Manager's Office Safe
1952
The code is the vintage year of the wine bottle on the manager's desk.
Dining Room Fireplace Puzzle
Lion → Snake → Goat
Order doesn't actually matter — you can press them in any sequence. There's a hint note in the room, or you can brute-force it.

Chapter 11: Uninvited (London Gala)

Maintenance Room Door Code
3256
Found on a laptop in the courtyard library.

Chapter 12: Knightfall (City of London)

Attic Safe
1493
Each digit is found by examining four items: diploma (1), painting (4), video game console (9), and scale model (3).

Chapter 15: Man of the Hour

Q-Lab Q-08 Armory Door Code
1324
The code is the top score displayed on the shooting range leaderboard.
Q-Lab Q-03 Pressure Test Cabinet
4397
The code is the license plate number of the car inside the test cabinet.

Chapter 16: Wave of the Future

Security Office — Archive Room Door Code
1864
Obtained from a safe downstairs, which you open using a key found in the Legal Department.
Pro Tip: You can enter these codes immediately when you reach the locked door or safe — no need to find the clues first. This is especially helpful for speed runs or cleanup via Chapter Select.
